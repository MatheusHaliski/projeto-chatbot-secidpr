# HU05 – Geração manual de combinações
**Como**: usuário comum  
**Posso**: gerar combinações de roupas manualmente  
**Para**: montar looks personalizados  

## Critérios de Aceitação
- Critério 1 – Combinação manual
  - **Dado que**: o usuário seleciona peças do guarda-roupa
  - **Quando**: cria uma combinação
  - **Então**: o sistema gera o novo look

- Critério 2 – Combinação com seleção insuficiente
  - **Dado que**: o usuário seleciona quantidade insuficiente de peças para formar um look
  - **Quando**: tenta criar uma combinação
  - **Então**: o sistema solicita a inclusão de mais peças antes de gerar o look

- Critério 3 – Salvamento da combinação manual
  - **Dado que**: o usuário gera uma combinação manualmente
  - **Quando**: opta por salvar o look criado
  - **Então**: o sistema registra a combinação para consultas futuras
