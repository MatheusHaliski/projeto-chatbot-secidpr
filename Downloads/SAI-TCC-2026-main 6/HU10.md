# HU10 – Compartilhamento em redes sociais
**Como**: usuário comum ou administrador  
**Posso**: postar esquemas de moda em redes sociais  
**Para**: compartilhar combinações  

## Critérios de Aceitação
- Critério 1 – Compartilhamento
  - **Dado que**: um esquema foi criado
  - **Quando**: o usuário escolhe compartilhar
  - **Então**: o sistema gera link ou imagem para publicação

- Critério 2 – Cancelamento do compartilhamento
  - **Dado que**: um esquema está pronto para compartilhamento
  - **Quando**: o usuário cancela a ação antes de publicar
  - **Então**: o sistema interrompe o fluxo sem realizar publicação

- Critério 3 – Compartilhamento de esquema inexistente
  - **Dado que**: não há esquema selecionado para compartilhamento
  - **Quando**: o usuário tenta compartilhar
  - **Então**: o sistema solicita a seleção de um esquema válido
