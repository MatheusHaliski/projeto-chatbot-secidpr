# HU13 – Registro de estatísticas de uso
**Como**: usuário comum ou administrador  
**Posso**: registrar estatísticas de uso das combinações  
**Para**: identificar combinações populares  

## Critérios de Aceitação
- Critério 1 – Registro estatístico
  - **Dado que**: um usuário interage com uma combinação
  - **Quando**: visualiza ou utiliza o esquema
  - **Então**: o sistema registra a interação

- Critério 2 – Consolidação por combinação
  - **Dado que**: existem múltiplas interações registradas em diferentes esquemas
  - **Quando**: consulta os dados estatísticos
  - **Então**: o sistema apresenta contagem de uso por combinação

- Critério 3 – Atualização contínua das estatísticas
  - **Dado que**: uma interação adicional acontece em um esquema já registrado
  - **Quando**: o evento é processado pelo sistema
  - **Então**: o sistema incrementa os dados estatísticos da combinação
