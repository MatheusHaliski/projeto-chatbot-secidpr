# HU12 – Acesso a lojas de marcas
**Como**: usuário comum ou administrador  
**Posso**: acessar lojas de marcas cadastradas no app  
**Para**: visualizar ou adquirir peças  

## Critérios de Aceitação
- Critério 1 – Acesso a loja
  - **Dado que**: o usuário seleciona uma marca
  - **Quando**: clica no link
  - **Então**: o sistema redireciona para o site externo

- Critério 2 – Marca sem link disponível
  - **Dado que**: o usuário seleciona uma marca sem URL cadastrada
  - **Quando**: tenta acessar a loja
  - **Então**: o sistema informa indisponibilidade de link para a marca

- Critério 3 – Retorno ao aplicativo após acesso externo
  - **Dado que**: o usuário foi redirecionado para o site da marca
  - **Quando**: retorna ao aplicativo
  - **Então**: o sistema mantém a sessão ativa e a navegação no app
