# HU16 – Configuração administrativa de catálogo
**Como**: usuário administrador  
**Posso**: configurar categorias de estilos, lojas e peças  
**Para**: manter o catálogo do sistema organizado  

## Critérios de Aceitação
- Critério 1 – Configuração administrativa
  - **Dado que**: o administrador acessa o painel
  - **Quando**: cria ou altera categorias, lojas ou peças
  - **Então**: o sistema salva as configurações

- Critério 2 – Validação de dados administrativos
  - **Dado que**: o administrador informa dados inválidos em uma configuração
  - **Quando**: tenta salvar alterações
  - **Então**: o sistema apresenta erro e impede o salvamento inválido

- Critério 3 – Reflexo das configurações no catálogo
  - **Dado que**: o administrador salva uma nova configuração válida
  - **Quando**: consulta o catálogo após a alteração
  - **Então**: o sistema exibe as categorias, lojas ou peças atualizadas
