# HU02 – Autenticação no sistema
**Como**: usuário comum, administrador ou varejista  
**Posso**: realizar autenticação no sistema  
**Para**: acessar minha conta com segurança  

## Critérios de Aceitação
- Critério 1 – Login válido
  - **Dado que**: o usuário informa email e senha corretos
  - **Quando**: solicita autenticação
  - **Então**: o sistema permite acesso à conta

- Critério 2 – Login inválido
  - **Dado que**: credenciais incorretas
  - **Quando**: tenta autenticar
  - **Então**: o sistema bloqueia o acesso

- Critério 3 – Campos obrigatórios de autenticação
  - **Dado que**: o usuário não informa email e/ou senha
  - **Quando**: solicita autenticação
  - **Então**: o sistema exibe mensagem de preenchimento obrigatório e não realiza login
