# HU08 – Cadastro de peças por fotografia
**Como**: usuário administrador  
**Posso**: cadastrar novas peças de roupa por fotografia  
**Para**: ampliar o catálogo do sistema  

## Critérios de Aceitação
- Critério 1 – Cadastro de peça
  - **Dado que**: o administrador envia a imagem da peça
  - **Quando**: confirma cadastro
  - **Então**: o sistema registra a peça

- Critério 2 – Imagem inválida para cadastro
  - **Dado que**: o administrador envia arquivo em formato não suportado
  - **Quando**: confirma cadastro
  - **Então**: o sistema rejeita o arquivo e solicita uma imagem válida

- Critério 3 – Peça cadastrada disponível no catálogo
  - **Dado que**: o administrador conclui cadastro de peça com sucesso
  - **Quando**: acessa o catálogo do sistema
  - **Então**: o sistema exibe a nova peça registrada
