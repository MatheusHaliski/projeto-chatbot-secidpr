# HU14 – Favoritar esquemas de moda
**Como**: usuário comum ou administrador  
**Posso**: favoritar esquemas de moda  
**Para**: acessar rapidamente combinações preferidas  

## Critérios de Aceitação
- Critério 1 – Favoritar
  - **Dado que**: o usuário visualiza um esquema
  - **Quando**: marca como favorito
  - **Então**: o sistema adiciona à lista de favoritos

- Critério 2 – Remover dos favoritos
  - **Dado que**: o esquema já está marcado como favorito
  - **Quando**: o usuário desmarca o favorito
  - **Então**: o sistema remove o esquema da lista de favoritos

- Critério 3 – Consulta de favoritos
  - **Dado que**: o usuário possui esquemas favoritados
  - **Quando**: acessa a lista de favoritos
  - **Então**: o sistema exibe apenas as combinações marcadas como favoritas
