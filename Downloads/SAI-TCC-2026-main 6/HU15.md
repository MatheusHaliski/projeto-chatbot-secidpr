# HU15 – Busca de roupas por filtros
**Como**: usuário comum ou administrador  
**Posso**: buscar roupas por filtros (estação, tipo, gênero)  
**Para**: encontrar peças adequadas  

## Critérios de Aceitação
- Critério 1 – Busca filtrada
  - **Dado que**: filtros são definidos
  - **Quando**: o usuário executa busca
  - **Então**: o sistema retorna itens correspondentes

- Critério 2 – Busca sem resultados
  - **Dado que**: filtros definidos não correspondem a peças cadastradas
  - **Quando**: o usuário executa busca
  - **Então**: o sistema informa que não há itens correspondentes

- Critério 3 – Combinação de múltiplos filtros
  - **Dado que**: o usuário define estação, tipo e gênero simultaneamente
  - **Quando**: executa a busca
  - **Então**: o sistema retorna somente peças que atendem a todos os filtros aplicados
