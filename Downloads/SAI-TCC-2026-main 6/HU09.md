# HU09 – Uso de manequim virtual 3D
**Como**: usuário comum  
**Posso**: utilizar um manequim virtual 3D  
**Para**: testar peças de roupa visualmente  

## Critérios de Aceitação
- Critério 1 – Visualização 3D
  - **Dado que**: o usuário seleciona uma peça
  - **Quando**: aplica no manequim
  - **Então**: o sistema exibe a peça no modelo 3D

- Critério 2 – Troca de peça no manequim
  - **Dado que**: o usuário já aplicou uma peça no manequim
  - **Quando**: seleciona uma nova peça para a mesma categoria
  - **Então**: o sistema atualiza a visualização 3D com a nova peça

- Critério 3 – Remoção de peça do manequim
  - **Dado que**: o usuário possui peça aplicada no manequim
  - **Quando**: remove a peça selecionada
  - **Então**: o sistema atualiza o modelo 3D sem a peça removida
