# HU04 – Consulta de esquemas salvos
**Como**: usuário comum ou administrador  
**Posso**: consultar esquemas de moda salvos  
**Para**: visualizar combinações criadas anteriormente  

## Critérios de Aceitação
- Critério 1 – Listagem
  - **Dado que**: o usuário possui esquemas salvos
  - **Quando**: acessa a lista
  - **Então**: o sistema exibe as combinações registradas

- Critério 2 – Lista sem registros
  - **Dado que**: o usuário não possui esquemas salvos
  - **Quando**: acessa a lista
  - **Então**: o sistema informa que não há combinações registradas

- Critério 3 – Acesso ao detalhe do esquema
  - **Dado que**: o usuário visualiza a lista de esquemas salvos
  - **Quando**: seleciona uma combinação específica
  - **Então**: o sistema exibe os detalhes do esquema escolhido
