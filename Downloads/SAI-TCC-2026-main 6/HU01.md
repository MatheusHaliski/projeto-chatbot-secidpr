# HU01 – Cadastro pessoal
**Como**: usuário comum, administrador ou varejista de marcas de roupa  
**Posso**: realizar cadastro pessoal  
**Para**: acessar meus recursos dentro do aplicativo  

## Critérios de Aceitação
- Critério 1 – Cadastro bem-sucedido
  - **Dado que**: o usuário informa corretamente todos os dados obrigatórios
  - **Quando**: solicita o cadastro
  - **Então**: o sistema realiza o registro e confirma a criação da conta

- Critério 2 – Cadastro com dados obrigatórios ausentes
  - **Dado que**: o usuário não preenche todos os dados obrigatórios
  - **Quando**: solicita o cadastro
  - **Então**: o sistema informa os campos pendentes e não conclui o registro

- Critério 3 – Cadastro com email já existente
  - **Dado que**: o usuário informa um email já cadastrado no sistema
  - **Quando**: solicita o cadastro
  - **Então**: o sistema bloqueia o novo registro e orienta a realizar autenticação ou recuperação de acesso
