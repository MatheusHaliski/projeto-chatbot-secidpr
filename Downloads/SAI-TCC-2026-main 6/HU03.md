# HU03 – Edição de perfil
**Como**: usuário comum ou administrador  
**Posso**: editar os dados do meu perfil  
**Para**: manter minhas informações atualizadas  

## Critérios de Aceitação
- Critério 1 – Atualização de perfil
  - **Dado que**: o usuário acessa edição de perfil
  - **Quando**: altera dados e salva
  - **Então**: o sistema atualiza as informações

- Critério 2 – Validação de campos no perfil
  - **Dado que**: o usuário preenche um campo em formato inválido
  - **Quando**: salva alterações do perfil
  - **Então**: o sistema apresenta erro de validação e mantém os dados anteriores

- Critério 3 – Persistência das alterações
  - **Dado que**: o usuário atualiza dados válidos do perfil
  - **Quando**: encerra sessão e acessa novamente a conta
  - **Então**: o sistema mantém os dados atualizados no perfil
