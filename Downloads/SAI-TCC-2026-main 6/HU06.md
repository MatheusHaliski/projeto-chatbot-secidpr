# HU06 – Geração de combinações com IA
**Como**: usuário comum  
**Posso**: gerar combinações usando inteligência artificial  
**Para**: descobrir novos estilos automaticamente  

## Critérios de Aceitação
- Critério 1 – Sugestão automática
  - **Dado que**: o usuário solicita sugestões
  - **Quando**: ativa a funcionalidade de IA
  - **Então**: o sistema apresenta combinações sugeridas

- Critério 2 – Sem dados suficientes para sugestão
  - **Dado que**: o usuário possui poucas peças cadastradas para análise
  - **Quando**: ativa a funcionalidade de IA
  - **Então**: o sistema informa limitação e orienta cadastrar mais peças

- Critério 3 – Nova sugestão sob demanda
  - **Dado que**: o usuário recebe sugestões iniciais de IA
  - **Quando**: solicita novas sugestões
  - **Então**: o sistema gera novas combinações automáticas
